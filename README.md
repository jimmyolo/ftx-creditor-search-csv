### How to use
You can download the binary build from [here](https://github.com/jimmyolo/ftx-creditor-search-csv/releases)
Download the csv file as well.
Put the binary & csv files into the same directory
```
$ tree -L 2
.
├── csv
│   ├── ftx_10.csv
│   ├── ftx_1.csv
│   ├── ftx_2.csv
│   ├── ftx_3.csv
│   ├── ftx_4.csv
│   ├── ftx_5.csv
│   ├── ftx_6.csv
│   ├── ftx_7.csv
│   ├── ftx_8.csv
│   └── ftx_9.csv
├── ftx_all.csv
└── linux-x64
```
